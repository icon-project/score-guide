Transaction¶

class Transaction(tx_hash: Optional[bytes] = None, index: int = 0, origin: Optional[Address] = None, timestamp: int = None, nonce: int = None)¶

Contains transaction info

hash¶

Transaction hash

index¶

Transaction index in a block

nonce¶

(optional) nonce of a transaction request. random value

origin¶

The account who created the transaction.

timestamp¶

Timestamp of a transaction request in microseconds This is NOT a block timestamp
